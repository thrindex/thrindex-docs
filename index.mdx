## Author, compile, run

```bash
pip install thrindex
```

```python
import thrindex as thx
import thrindex.snn as snn

model = snn.Sequential(
    snn.Dense(700, 512),
    snn.LIF(tau_mem=20.0, threshold=0.3),
    snn.Dense(512, 20),
    snn.LIF(tau_mem=20.0, threshold=0.3),
)

thx.compile(model, "model.thx")
```

```bash
thrindex run model.thx
```

## THRINDEX Certified

Hardware backends earn the badge by passing the conformance harness against `CONFORMANCE_ENVELOPE_V0` — a derived bound on divergence from the float reference simulator.

| Metric | Threshold | Meaning |
|---|---|---|
| `T_mean` | ≤ 0.020 | Mean spike-count error ≤ 2% |
| `T_max` (p99) | ≤ 0.130 | Worst-neuron error ≤ 13% |
| `P_pred` | ≥ 0.900 | ≥ 90% of predictions agree |

[Read the conformance spec →](/conformance/envelope-v0)

## What THRINDEX is not

- **Not a general-purpose framework.** Spiking neural networks on neuromorphic hardware — specifically.
- **Not a black box.** Every run prints a structured transcript: spike counts, syn-ops, energy estimate.
- **Not an abstraction over hardware.** A deterministic model of what happens on the target, with a conformance bound on divergence.
