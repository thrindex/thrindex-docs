THRINDEX is an open-source toolchain for spiking neural networks. You author models in Python using familiar PyTorch-style primitives, compile them to a sealed artifact, and run them through a behavioral simulator or a neuromorphic hardware backend — with the same command, the same output format, and the same conformance guarantee.

```bash
pip install thrindex
```

```python
import thrindex as thx
import thrindex.snn as snn

model = snn.Sequential(
    snn.Dense(700, 512),
    snn.LIF(tau_mem=20.0, threshold=0.3),
    snn.Dense(512, 20),
    snn.LIF(tau_mem=20.0, threshold=0.3),
)

thx.compile(model, "model.thx")
```

```bash
thrindex run model.thx
```

## What THRINDEX is not

- It is not a general-purpose deep learning framework. It targets spiking neural networks specifically.
- It does not hide the spike raster. The output of every run is a structured transcript you can inspect, diff, and test.
- It does not abstract away the hardware. It gives you a deterministic model of what will happen on the target, and a conformance guarantee that bounds the divergence from that model.
